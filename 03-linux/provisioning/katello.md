# Katello

## What is Katello?

Katello is a plugin for Foreman that adds content and subscription management capabilities.

## Main Features

- Repository synchronization
- Content Views
- Lifecycle Environments
- Package management
- Patch management
- Subscription management

## Architecture

Katello uses:

- Pulp
- Candlepin
